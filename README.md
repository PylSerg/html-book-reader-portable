# AshtiEmile Reader

**AshtiEmile Reader** is a premium, standalone, and highly portable HTML book reader. Designed for users who value simplicity, privacy, and a refined reading experience, this application runs entirely in your web browser without the need for a backend or installation.

## 🌟 Key Features

### 📖 Exceptional Reading Experience
*   **Reading Modes:** Switch between classic continuous scrolling and a modern paginated (column-based) layout with horizontal page flipping.
*   **Progress Tracking:** Automatically saves your reading position for every book in your library.
*   **Typography Control:** Complete control over font family, size, line height, weight (Bold), and style (Italic).
*   **Smart Layout:** Adjustable horizontal and vertical margins to fit any screen size or reading preference.
*   **Hyphenation:** Built-in support for text hyphenation for beautifully justified paragraphs.
*   **Image Gallery:** Integrated viewer for book illustrations with gallery navigation support.

### 📚 Library Management
*   **Standalone Library:** Store your entire collection locally using your browser's IndexedDB.
*   **Batch Import:** Add multiple `.html` books simultaneously.
*   **Export/Import:** Back up your entire library, including books and reading progress, to a single `.json` file for easy transfer between devices.
*   **Search & Filter:** Find your books instantly using the real-time search field.
*   **Automatic Organization:** Books are automatically grouped by author for a clean, organized view.

### 🎨 Customization & UI
*   **Color Themes:** Full control over background and text colors via integrated color pickers.
*   **Dynamic Typography:** Access a curated selection of Google Fonts (loaded dynamically when online), sorted alphabetically with support for Cyrillic scripts.
*   **Rich Toolbar:** Stay informed with an integrated clock, battery indicator, and precise reading progress percentage.
*   **Custom Scrollbar:** A unique, progress-aware scrollbar that visualizes your depth in the book.
*   **Full-Screen Mode:** Immerse yourself in your reading with a single click.

### 🌍 Global Reach
*   **Multilingual Support:** Fully localized interface in over 50 languages, including English, Ukrainian, French, German, Spanish, Arabic, Japanese, and many more.

## 🛠️ Technical Details

*   **Zero Dependencies:** Built with pure Vanilla JS, CSS, and HTML. No external frameworks or servers required.
*   **Offline First:** Once loaded, the reader works entirely offline.
*   **Storage:** Uses the modern `IndexedDB` API for robust, high-capacity local storage.
*   **Single File:** The entire application is contained within a single HTML file, making it the ultimate portable tool.

## 🚀 Getting Started

1.  **Download** the `AshtiEmile_Book_Reader__v1-0.html` file.
2.  **Open** the file in any modern web browser (Chrome, Firefox, Safari, Edge).
3.  **Import Books:** Click the menu icon (≡) in the top-left corner, select "Add Book", and choose your HTML book files.
4.  **Enjoy Reading!** Your progress and settings are saved automatically.
